详细描述

这是一个使用log的工具

具有如下功能

  • 设置到不同的输出设备
  • 改变日志的文本格式
  • 提供日志等级自动过滤功能

void tinyLogger::format ( char *  str  ) 

指定格式化字符串，以后的信息都以此格式进行输出

参数:
str 整数(keyword)，其中整数可以是正也可以是负，分别代表左右对齐，例：%25(asctime) 或 %-25(asctime)。 关键字可以是:
  • asctime 当前时间
  • filename 当前文件名
  • lineno 当前行号
  • funame 坐在函数名称

void tinyLogger::install_msgHandler ( LogMsgHandler  fn  )  [inline]

这个函数用来注册消息通知函数,当log运行的时候，此注册函数将会被调用

fn的原型: typedef void (*LogMsgHandler)( const char* type, const char* msg ); type将会被设置成info, warning, error, debug
